## 概要(模擬案件)<br>
・VBA学習開始約一週間後に作成したプログラムです。<br>
　（VBA学習：2026年2月25日～　制作期間：2026年3月6日～）<br>
・依頼者：	カラオケ会の幹事(個人でも確認)<br>
　依頼内容：	開催後の歌リスト作成を効率化したい。<br>
 　　　　　　歌リスト作成後、リスト一覧に反映させたい。<br>
　現状課題：	カラオケ機種により入力形式が異なる。
　　　　　　　人ごとの整形に時間がかかり、現在は手作業で1時間ほどかかっている。<br>
　　　　　　　リスト一覧に反映させるのは、かなり時間がかかっている。
　　　　　　　情報をネット検索して追加したりもしている。<br>
